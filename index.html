HTML

  • The head element contains information about the webpage
  • Inside the head we can use elements such as title, style, base, link, meta and script.
  • The body element represents the visible content shown to the user.
  • !DOCTYPE html is a declaration and it tells the browser what document type it should expect.
  • There are some common element types that are use to divide and describe the page: meta-related, content division and semantic.
